Practicando Git y GitHub con Git Bash

Esta página es parte de una práctica para entender cómo Git y GitHub funcionan juntos.

Pasos realizados:

  • Creación de un repositorio local con git init
  • Primer commit con git commit
  • Enlace al repositorio remoto en GitHub
  • Push de los cambios con git push
  • Modificación del archivo index.html para reflejar la práctica
